## [0.1.0] - 2025-12-31
- Initial “Thinking in Code” stack: FastAPI API service, worker pipeline, shared contracts, and podcast engine.
- PDF ingestion with chunking + citation preservation.
- LLM-driven script generation with strict guardrails, fallback script builder, and quality scoring.
- Optional embeddings/vector stores (Chroma/FAISS/local) and TTS/audio assembly with loudness/quality reports.
- Redis/file/dir queues, SSE progress, metrics, tracing, and API key auth.
- Web UI with drag/drop uploads, progress feed, artifact gallery, and playback hooks.
- Dockerfiles + docker-compose + CI workflow.
